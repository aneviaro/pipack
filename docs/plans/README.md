# Plans

Implementation plans live here. Each plan tracks completion via its `Status:`
field in the **Source Spec** section (`Approved` → `Completed`).

When a plan is `Completed`, move its file into `completed/`:

- Active plans: `docs/plans/<plan>.md`
- Completed plans: `docs/plans/completed/<plan>.md`

When moving a plan, update any inbound references to its old path.
